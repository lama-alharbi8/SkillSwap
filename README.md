# SkillSwap
# Project Name: 
SkillSwap

# Project Description:
A skill-exchange platform where users can offer their abilities and request help with skills they need. The system supports user profiles, smart skill matching, circular exchanges, and admin management of skills, all presented with a clean, interactive interface.

# Features list:
- User authentication (signup, signin, logout).
- Users can select multiple offered and needed skills.
- Staff-only panel to add new skills.
- Offered Skill model with description, availability, hourly rate, and status.
- Needed Skill model with description, urgency, max hourly rate, and status.
- Exchange Chain where 3 users can exchange skills.
- Smart Matching that finds best matches based on hourly rate.
- Clean hero section with CTA buttons.
- User Statistics page showing offered, needed, and exchanges numbers.
- User feedback and notifications throughout the platform.
